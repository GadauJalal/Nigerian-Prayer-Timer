ADHAN SOUND FILE REQUIRED
=========================

Please add an Adhan sound file named "adhan.mp3" to this directory.

The file should be:
- Named: adhan.mp3
- Format: MP3
- Duration: 30 seconds to 2 minutes (recommended)
- Quality: Good audio quality, not too large (< 5MB recommended)

You can download Adhan MP3 files from Islamic websites or use your own recording.

Once you have the adhan.mp3 file, place it directly in the assets/ folder.

The notification system will automatically use this sound when prayer time notifications are triggered.
